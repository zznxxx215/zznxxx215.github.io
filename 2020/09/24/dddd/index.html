利用github+git+hexo搭建个人博客

准备

注册GitHub账号

官网

安装Git

官网 注意下载与操作系统对应版本

安装node.js

官网 注意下载与操作系统对应版本

检测安装是否成功：

打开cmd输入：
node -v
npm -v
如果安装成功会显示相应的版本号

Hexo安装

在磁盘中创建一个文件夹例如blog，在该文件夹中鼠标右击点开git base here
输入 npm install hexo-cli -g 来安装hexo
再输入 npm install hexo-deployer-git –save 把hexo部署到 git page deployer上
（因为我已经安装过了所以不演示)
在xxx文件夹中创建一个文件夹xxxblog右击打开git base here
输入hexo init 把hexo初始化在此文件夹中


git 中输入 hexo generate
hexo s

上图表示hexo成功在4000端口运行
浏览器中输入 http://localhost:4000/ 进行浏览

在Github部署自己的博客

在Repository name中填写自己的github账户名+ .github.io(例：zznxxx215.github.io),其他默认点击create repositor创建库

在xxx文件夹下git base here输入
ssh-keygan -t rsa -C “”(引号中填写自己的邮箱地址)
看到Enter file in which to save the key (/c/Users/you/.ssh/id_rsa):
Enter passphrase (empty for no passphrase):
Enter same passphrase again:按回车
之后会显示
Your identification has been saved in /c/Users/you/.ssh/id_rsa.
Your public key has been saved in /c/Users/you/.ssh/id_rsa.pub.
The key fingerprint is:

The key’s randomart image is:

继续输入clip < ~/.ssh/id_rsa.pub（把公钥复制到剪贴板上）


title随便填

git config –global user.name “此处填你的git用户名（可以填一样的github用户名方便记）”
git config –global user.email “此处填你的邮箱”
